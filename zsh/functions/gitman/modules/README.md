# Modules gitman

Ce répertoire contient les modules du gestionnaire gitman.

## Structure

```
gitman/
├── core/
│   └── gitman.zsh          # Script principal
├── modules/                    # Modules fonctionnels
├── utils/                       # Utilitaires
├── config/
│   └── gitman.conf         # Configuration
└── install/                     # Scripts d'installation
```
