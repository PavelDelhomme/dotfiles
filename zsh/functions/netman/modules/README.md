# Modules netman

Ce répertoire contient les modules du gestionnaire netman.

## Structure

```
netman/
├── core/
│   └── netman.zsh          # Script principal
├── modules/                    # Modules fonctionnels
├── utils/                       # Utilitaires
├── config/
│   └── netman.conf         # Configuration
└── install/                     # Scripts d'installation
```
