# Modules miscman

Ce répertoire contient les modules du gestionnaire miscman.

```
miscman/
├── core/

├── modules/                    # Modules fonctionnels
├── utils/                       # Utilitaires
├── config/
│   └── miscman.conf         # Configuration
└── install/                     # Scripts d'installation
```
