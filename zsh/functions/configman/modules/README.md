# Modules configman

Ce répertoire contient les modules du gestionnaire configman.

## Structure

```
configman/
├── core/
│   └── configman.zsh          # Script principal
├── modules/                    # Modules fonctionnels
├── utils/                       # Utilitaires
├── config/
│   └── configman.conf         # Configuration
└── install/                     # Scripts d'installation
```
