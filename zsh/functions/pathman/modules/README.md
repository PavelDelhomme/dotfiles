# Modules pathman

Ce répertoire contient les modules du gestionnaire pathman.

## Structure

```
pathman/
├── core/
│   └── pathman.zsh          # Script principal
├── modules/                    # Modules fonctionnels
├── utils/                       # Utilitaires
├── config/
│   └── pathman.conf         # Configuration
└── install/                     # Scripts d'installation
```
